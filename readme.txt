--How to change icons and add file paths--
1. To change Icons, just download the desired icons form Remix Icons["https://remixicon.com/"] in png format and paste it in "C:\Users\<Your_Username>\Documents\Rainmeter\Skins\MonoDock\@Resources\icons". Then refresh the skin !
